# 🛡️ Ads-Manager Obsidian Vault

Hệ thống quản lý dự án phần mềm quảng cáo Facebook.

## 📂 Cấu trúc thư mục
- **[[00-Meta/Templates|Templates]]**: Chứa các mẫu ghi chú (Project, Feature, Bug).
- **[[10-Projects]]**: Các dự án con và Master Note [[10-Projects/Ads-Manager-Core|Ads-Manager-Core]].
- **[[20-Tasks]]**: Danh sách Task (Sử dụng plugin Checklist hoặc Tasks nếu cần).
- **[[30-Notes]]**: Brainstorming và kiến thức kỹ thuật.
- **[[40-Resources]]**: Tài liệu tham khảo, thiết kế, API.

## 🛠️ Hướng dẫn nhanh
1.  **Tạo ghi chú mới**: Nhấn `Ctrl + N`.
2.  **Áp dụng mẫu**: Nhấn `Ctrl + P`, gõ `Insert template` và chọn mẫu mong muốn.
3.  **Xem tiến độ**: Mở [[10-Projects/Ads-Manager-Core|Ads-Manager-Core]].
